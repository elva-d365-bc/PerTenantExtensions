    Overview

    Report Customer - G/L Account Trial Balance
    Report Vendor - G/L Account Trial Balance

    Report Customer - G/L Account Trial Balance is a ELVA custom report, integrated with Dynamics 365 Business Central.

    Report Vendor - G/L Account Trial Balance is a ELVA custom report, integrated with Dynamics 365 Business Central.

    Benefits

    Possibility to obtain a Report Customer - G/L Account Trial Balance or Report Vendor - G/L Account Trial Balance

    Assumptions and limitations

    Latvian Localization must be installed Latvian Localization.

    How to get started

    Report Report Customer - G/L Account Trial Balance available by typing in the search Customer - G/L Account Trial Balance
    Report Report Vendor - G/L Account Trial Balance available by typing in the search Vendor - G/L Account Trial Balance

    How to set up

    No specific settings is required.

    How to use

    In the section Options must be selected Start Date, End Date.
    Print Details – If the option is checked, each transaction will be printed.
    In the section Filter filters should be set up if necessary, example. if necessary, filters should be installed, e.g. Customer/Vendor, G/L Account.

    The starting balance in the selected G/L Account for the specific Customer, the Change in the selected period and the final balance are displayed.
    Example of a printout
    The opening balance in the selected G/L account for the specific supplier, the Change in the selected period, and the closing balance are displayed.
