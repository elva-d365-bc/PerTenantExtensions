    Overview

    Trial Balance by Accounts

    Report Trial Balance by Accounts ir ELVA custom report, integrated with Dynamics 365 Business Central.

    Benefits

    Dynamics 365 Business Central creates compound account correspondence, where postings use three or more accounts, so it is not possible to obtain account correspondence reports that show the postings in the form of simple account correspondence. The Trial Balance by Accounts report displays all postings involving the selected ledger account for the specified period.
    The report Trial Balance by Accounts can be viewed both with details for each posting and in totals for the period.

    Assumptions and limitations

    Latvian Localization must be installed Latvian Localization.

    How to get started

    Report Trial Balance by Accounts available by typing in the search Trial Balance by Accounts

    How to set up

    No specific settings is required.

    How to use

    In the section Options if it is necessary to show the sums together in the report by ledger accounts, it should be noted using the slider Show Only Total Amounts

    In the section Filter G/L Account
    No the ledger account number or numbers for which the report is to be prepared must be specified.
    Date filter must be selected Start Date, End Date.
    Additional filters can be added as needed.
