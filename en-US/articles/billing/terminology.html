    Terminology

    Basic Terms

    English Term Description
    Contract Main document registering agreement with customer or vendor for service provision
    Contract No. Unique contract identifier
    Contract Status Current contract state (Active, Closed, Interrupted, Inactive)
    Service Catalog element defining provided service
    Service Code Unique service identifier
    Service Variant Service variation (e.g., different quality levels)
    Object Physical or logical object for which services are provided
    Object Code Unique object identifier
    Object Category Object classification
    Partner Customer or vendor
    Partner Source Whether partner is customer or vendor
    Partner No. Partner number in system

    Calculation Terms

    English Term Description
    Calculation Worksheet Tool for service calculation
    Worksheet Template Worksheet type or category
    Worksheet Batch Specific work session or grouping
    Calculation Line Individual calculation entry in worksheet
    Calculation Period Period for which calculation is performed
    Calculation Ledger Entry Approved calculation entry
    Quantity Service quantity
    Quantity Source How quantity is determined (manual, calendar, object)
    Quantity Coefficient Multiplier for quantity adjustment
    Base Quantity Initial quantity before coefficient application

    Price and Discount Terms

    English Term Description
    Unit Price Price per one unit of measure
    Base Unit Price Initial price before discount
    Price Including VAT Whether price includes value added tax
    Price Source Where price is taken from (contract, service, price list)
    Price Coefficient Multiplier for price adjustment
    Price Date Date for which price is determined
    Total Price Price × Quantity
    Discount Discount amount or percentage value
    Discount, % Discount percentage rate
    Discount Amount Discount amount in monetary terms
    Discount Source Where discount is taken from
    Discount Usage How discount is applied
    Base Unit Price Incl. Discount Price after discount application
    Total Price Incl. Discount Total amount after discount application
    Min Amount Minimum allowable amount
    Max Amount Maximum allowable amount
